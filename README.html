Introduction

The project now called DragonFiesta is a continued branch of the preivous Zepheus called project, aiming for a complete Fiesta emulator.

Setup and Running

Setup

1st Make sure that your copy of the project compiles without any errors. 2nd You need to setup a MySQL Database, add a Scheme and run all the Scripts found in the sqlbase folder on it. 3rd You'll need and want to Edit the configuration file, matching your setup.

Edit any Mysql propertes, so they match your settings.
Also, you'll want to edit the public IP's: World.IP and Zone.IP

Running

Just run all the *.exe files in the Run folder, they will give an output on any error occurance.
You'll also want to try and connect to your Server, if you can't do so, go and check the Public-IP's in your Configuration file.

GM Commands

There are a few GM-Commandos which are currently supported. The list keeps getting updated.

  • &anim <AnimationID> Let you Character make the Animation with the give ID.
  • &map <MapName> Ports you to the given map
  • &warpnpc <NPC Name> Teleports you to the given NPC
  • &find <Character Name> Displays the coordinates of the given Character, if online
  • &warp <Character Name> Teleports you to the given Character, if online
  • &tele <X> <Y> teleports you to the given Position.
  • &pos displays you current position
  • &heal heals your character.
  • &mount <Mount ID> Mounts you on the given mount.
  • &umount unmounts you.
