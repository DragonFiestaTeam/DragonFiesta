[Release] Fiesta Database Extractor
I thought it would be usefull to create a full database (QuestData.shn supported!) for Fiesta fansites / Private servers.

This tool can convert most of the SHN Files (ressystem) into (My)SQL readable scripts / insert queries. This way you can use them in PHP.

It allows you to multi-select the SHN Files and will create a .SQL file supported by most of the MySql versions.

Enjoy!~
